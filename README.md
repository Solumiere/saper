# 💣 Сапёр (Minesweeper)

Полноценная реализация классической игры **Сапёр** на Python с графическим интерфейсом на `tkinter`.

![Python](https://img.shields.io/badge/Python-3.8%2B-blue?logo=python&logoColor=white)

## ✨ Возможности

- 🎚 **Выбор сложности**: Новичок (9×9, 10 мин), Любитель (16×16, 40 мин), Профи (16×30, 99 мин) и **своя игра** с произвольными параметрами.
- 🎨 **Красивый тёмный визуал** с классическими цветными цифрами.
- 🙂 **Кнопка-смайлик** для перезапуска, **таймер** и **счётчик оставшихся мин**.
- 🛡 **Первый клик всегда безопасен** — вы не подорвётесь на первом ходу.
- 💧 **Автораскрытие** пустых областей.
- 🚩 **Флажки** (правая кнопка мыши) и **"аккорд"** (двойной клик по открытой цифре для быстрого раскрытия соседей).

## 🚀 Запуск

Требуется Python 3.8+ (модуль `tkinter` входит в стандартную поставку Python).

```bash
python saper.py
```

> На некоторых дистрибутивах Linux может потребоваться установить tkinter отдельно:
> ```bash
> sudo apt install python3-tk
> ```

## 🎮 Как играть

- **Левая кнопка мыши** — открыть клетку.
- **Правая кнопка мыши** — поставить/снять флажок.
- **Двойной клик** по открытой цифре — открыть всех соседей, если рядом достаточно флажков.
- Меню **«Игра»** — выбор уровня сложности или своей игры.

Цель — открыть все клетки без мин. Удачи! 🍀
